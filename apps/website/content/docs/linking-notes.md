Notes in Companion are plain Markdown, and every note can link to any other. Those links are what turn a pile of notes into a connected body of knowledge.

## Backlinks

Type `[[` and search for a note's title to link to it. The connection is two-way: the note you linked now lists this one under its **linked notes**, so you can trace an idea in either direction.

## Markdown

Format as you type. Headings, lists, quotes, code, and emphasis all render live:

- `## Heading` for a section
- `- item` for a bullet
- `**bold**` and `*italic*` for emphasis

> **Tip:** Add `#tags` anywhere to gather related notes across folders.
